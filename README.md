A simple and fun browser-based click game built using HTML, CSS, and JavaScript.

## 🌐 Live Demo

Play the game here:

https://darshan-raj-christeen.github.io/clickz-game-v2/

---

## 📖 About the Game

Clickz Game is a reaction-based game where players must click a moving target to earn points while avoiding wrong clicks.

The game includes:

* Score tracking
* Life management system
* Rewarded ad simulation
* Continue after Game Over feature
* Progressive Web App (PWA) support
* Mobile-friendly design

---

## 🚀 Features

### 🎯 Scoring System

* Click the moving target to earn points.
* Each successful click increases the score.

### ❤️ Life System

* Players start with 5 lives.
* Clicking outside the target reduces one life.

### 📺 Reward Ad System

* Watch a simulated advertisement to gain an extra life.
* Maximum life limit is enforced.

### 🔄 Continue Feature

* When lives reach zero, players can continue by watching a rewarded advertisement.
* Continuing grants additional lives while preserving progress.

### 📱 Progressive Web App (PWA)

* Installable on supported devices.
* Works directly from the browser.
* Includes manifest configuration for app packaging.
